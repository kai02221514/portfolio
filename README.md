### ユ活（入浴管理PWA）

<img width="870" height="483" alt="SS 7" src="https://github.com/user-attachments/assets/be3d586d-445c-4c9e-b123-de0e044d2c17" />

- リポジトリ: https://github.com/2509-gabaithon/2509-gabaithon
- サイト: https://yukatsu.onrender.com/

- 概要：  
  2025年開催のガバイソンで開発した温泉入浴に付加価値をつけるPWAアプリ

- 開発背景：  
  開催場所であった佐賀県は温泉で有名だったが、温泉利用は継続的な行動になりにくく、記録や動機付けが不足しているという課題があった。

- 工夫点：
  - パートナー機能による行動の動機付け設計
  - スタンプ機能により利用履歴を可視化し、継続意欲を向上
  - PWA化によりネイティブアプリに近い操作性を実現

- 使用技術：  
  Next.js / TypeScript / Supabase / Figma / Figma AI

- 担当領域：  
  フロントエンド実装およびPWA対応
